# QSR Daily Ops
**v1.0.1 · McPherson AI · San Diego, CA**

AI-powered daily operations system for QSR managers: shift planning, execution checklists, labor control, and real-time operational visibility.

---

## Overview

QSR Daily Ops is a practical restaurant operations skill built for quick-service managers, assistant managers, and owner-operators who need tighter daily control of the business.

It helps operators stay ahead of the day instead of reacting to it by organizing operational priorities into a clear management workflow.

This skill is designed to support:

- Daily shift readiness
- Labor awareness
- Operational checklists
- Manager focus and prioritization
- Mid-shift correction
- End-of-day control

It is built from real operating experience inside high-volume QSR environments.

---

## What It Does

QSR Daily Ops functions as a structured operations assistant for store leadership.

It can help managers:

- Build a daily operational game plan
- Prioritize the most important tasks for the shift
- Track labor pressure and staffing gaps
- Organize opening, mid-shift, and closing responsibilities
- Identify execution risks before they become service failures
- Improve consistency across managers and shifts
- Maintain visibility on throughput, staffing, readiness, and store condition

Rather than acting like a generic chatbot, this skill is designed to think like a QSR operator.

---

## Core Use Cases

### 1. Shift Planning
Create a clear plan for the day based on staffing, expected sales, labor pressure, and key operational priorities.

### 2. Execution Checklists
Organize responsibilities into practical opening, rush, recovery, and closing actions.

### 3. Labor Discipline
Keep managers aware of labor creep, scheduling drift, idle coverage, and unnecessary overstaffing.

### 4. Mid-Shift Adjustment
Spot developing issues early and recommend where to refocus effort.

### 5. Leadership Support
Help newer managers think more structurally about running a shift and balancing people, service, and cost control.

## Why It Exists

Most restaurant tools focus on reporting after the fact.

QSR Daily Ops is built to improve decision-making during the day, while managers still have time to act.

The goal is simple:

**better daily execution, tighter labor control, and fewer preventable operational misses.**
